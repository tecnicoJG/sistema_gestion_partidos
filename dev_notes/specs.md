## Score tracking system

## Components

- Server + optional sotrage server
- Court Companion (court management hub) (SBC)
- 4x score sensors (2 per team) (BT / RS-485 using M12 5P connector) daisy chain
- Wireless touch HMI, android tablet kiosk + charging stand
- Court display (42" TV, HDMI)
- IP Gameplay recording camera

## Match Management System Features

- WiFi AP mode / WiFi network mode
- BT-5.0 for remote points sensor
- 2x RS-485 M12 connectors for points sensor
- Local Web UI for remote management
- 2x NO/NC 16A relays

- Web UI features

  - Admin user (password protected device configurations)
  - Staff user (password protected match configurations)
  - kiosk user (authorized linked devices)
  - Guest user (publicly accessible password, interfaces with limited control)
  - Sport options: padel, tennis, football, basketball, etc..

- usage flow

  **installation**

  1. Connect USB-C device power
  2. Connect to device hotspot - Hold device reset button if unavailable
  3. Open browser to device IP/hostname
  4. Complete WiFi connection configuration. Device will reboot - (optional)
  5. Complete Cloude Connection configuration and/or
  6. Enter new Admin user password
  7. Complete device configuration
